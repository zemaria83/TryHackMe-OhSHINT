# OSINT Investigation – OhSINT (TryHackMe)

##  Overview
This project documents an OSINT investigation conducted as part of the OhSINT room on TryHackMe.

The objective was to extract as much information as possible from a single image using open-source intelligence techniques.

---

##  Objectives
- Identify the target user
- Determine location
- Discover WiFi network information
- Extract email and online presence
- Identify travel history
- Recover hidden password

---

##  Methodology

The investigation followed a structured OSINT workflow:

1. Metadata extraction using ExifTool  
2. Username pivoting and GitHub analysis  
3. Social media investigation (Twitter/X)  
4. WiFi analysis using Wigle  
5. Website investigation  
6. Source code inspection  

---

##  Tools Used
- ExifTool  
- Google Search  
- GitHub  
- Twitter/X  
- Wigle.net  
- Google Maps  
- Browser Developer Tools   

---

##  Challenges
- Correlating multiple sources of information  
- Identifying relevant data among noise  
- Discovering hidden content in source code  

---

##  Key Takeaways
- Small metadata details can lead to full identification  
- Pivoting between platforms is essential  
- Public WiFi data can expose sensitive information  
- Source code inspection can reveal hidden secrets
